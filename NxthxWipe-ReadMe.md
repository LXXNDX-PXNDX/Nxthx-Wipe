# NetherWipe – macOS Netzwerk-Zerstörungstool

**Autor:** CygnusX  
**Version:** 1.0  
**Zweck:** Zerstört alle Geräte in einem WiFi-Netzwerk durch DoS, ARP-Spoofing, Deauth und Router-Reset.

## Installation

1. Homebrew installieren (falls nicht vorhanden)
2. `brew install hping3 arpspoof aircrack-ng python-tk`
3. `pip3 install scapy requests`

## Nutzung

```bash
sudo python3 NetherWipe.py
